# bim-viewer

BIM Viewer auf Basis von xeokit-bim-viewer / xeokit SDK.

## Zweck
Dieses Repository enthält den öffentlich offengelegten Quellcode des BIM Viewers.

## Enthalten
- Viewer-Code
- Branding / Styling
- Build-Artefakte
- Leere Projektstruktur ohne produktive Modelldaten

## Nicht enthalten
- produktive Projektdaten
- interne XKT-Dateien

## Lizenz
Dieses Projekt basiert auf [xeokit-bim-viewer / xeokit SDK](https://github.com/xeokit/xeokit-bim-viewer).
Lizenzhinweise und Quelloffenlegung gemäss AGPLv3.
